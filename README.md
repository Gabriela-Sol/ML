# Proyecto de IA - Machine Learning

Este repositorio contiene un proyecto de inteligencia artificial para cargar datasets y entrenar modelos de machine learning utilizando varios notebooks de Jupyter.

## Estructura del Proyecto

- `datasets/`: Carpeta para almacenar los datasets utilizados en el proyecto.
- `notebooks/`: Notebooks de Jupyter para análisis de datos, preprocesamiento y entrenamiento de modelos.
- `src/`: Código fuente adicional, como scripts de utilidad o módulos personalizados.
- `models/`: Modelos entrenados guardados.
- `requirements.txt`: Lista de dependencias de Python necesarias.
- `.gitignore`: Archivos a ignorar en el control de versiones.

## Instalación

1. Clona este repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`.
3. Abre los notebooks en Jupyter o VS Code para ejecutar el código.

## Uso

- Carga tus datasets en la carpeta `datasets/`.
- Utiliza los notebooks en `notebooks/` para explorar datos y entrenar modelos.
- Guarda los modelos entrenados en `models/`.
